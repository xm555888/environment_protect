# My Digital Biome - Landing Page

## 🌍 项目概述

我们成功创建了一个精美的环保主题landing page，将原有的生态模拟游戏转化为一个综合性的环保教育门户网站的入口。

## ✨ 主要特性

### 🎨 视觉设计
- **现代化设计风格** - 采用渐变色彩和自然主题
- **响应式布局** - 完美适配桌面端和移动端
- **流畅动画效果** - 包含轮播、悬停效果和平滑滚动
- **环保色彩方案** - 绿色系主色调体现环保属性

### 🧭 导航功能
- **固定顶部导航** - 透明背景，毛玻璃效果
- **平滑滚动** - 点击导航链接平滑滚动到对应区域
- **移动端菜单** - 响应式汉堡菜单设计

### 📱 页面结构

#### 1. Hero区域（英雄区域）
- **自动轮播** - 3个不同主题的内容轮播
- **实时数据展示** - 环保相关的关键指标
- **双CTA按钮** - "开始生态探索之旅" 和 "了解更多"
- **视觉吸引力** - 大型emoji图标和渐变文字

#### 2. 功能特色区域
- **4个核心功能** - 互动生态模拟器、环保数据可视化、个人环保工具、科学知识库
- **卡片式布局** - 悬停效果和图标设计
- **渐变背景** - 每个功能卡片都有独特的渐变色

#### 3. 影响力数据区域
- **统计数据展示** - 用户数量、减少碳排放、覆盖地区、环保项目
- **绿色背景** - 突出环保主题
- **图标配合** - 每个数据都有对应的图标

#### 4. 使命陈述区域
- **双栏布局** - 左侧文字说明，右侧产品展示
- **三大支柱** - 寓教于乐、科学严谨、行动导向
- **产品预览** - My Digital Biome的精美展示卡片

#### 5. 最终CTA区域
- **深色背景** - 营造紧迫感和专业感
- **大号标题** - 强烈的行动召唤
- **双按钮设计** - 主要和次要行动选项

#### 6. 页脚
- **简洁设计** - 品牌信息和核心价值观
- **版权信息** - 完整的法律声明

## 🔧 技术实现

### 技术栈
- **Next.js 16.0.1** - React框架
- **TypeScript** - 类型安全
- **Tailwind CSS** - 样式框架
- **Lucide React** - 图标库

### 关键功能
- **自动轮播** - 5秒间隔的内容切换
- **平滑滚动** - CSS scroll-behavior和JavaScript结合
- **响应式设计** - 移动端优先的设计理念
- **性能优化** - 图片优化和动画性能优化

### 文件结构
```
src/
├── components/
│   └── landing/
│       └── LandingPage.tsx     # 主要landing page组件
├── styles/
│   └── landing.css             # landing page专用样式
├── app/
│   ├── page.tsx                # 首页（使用LandingPage组件）
│   └── game/
│       └── page.tsx            # 游戏页面
```

## 🚀 使用方法

### 开发环境
```bash
npm run dev
```
访问 http://localhost:3000 查看landing page

### 页面导航
- **首页** - `/` - 显示landing page
- **游戏页面** - `/game` - 原有的生态模拟器
